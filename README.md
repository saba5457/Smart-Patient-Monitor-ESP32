# 🏥 Smart Patient Monitor — ESP32

IoT-based real-time patient health monitoring system.

## Sensors Used
- MAX30105 → Heart Rate + SpO2
- DS18B20 → Body Temperature
- WiFi → Send data to Dashboard

## Tech Stack
- ESP32 IoT Controller
- Arduino IDE
- LangChain Libraries

## Features
- Real-time heart rate monitoring
- Blood oxygen level (SpO2) tracking
- Body temperature monitoring
- WiFi data transmission
- Alert system for abnormal readings

## How to run
1. Install Arduino IDE
2. Install ESP32 board
3. Upload smart_patient_monitor.ino
4. Connect sensors as per pin configuration
